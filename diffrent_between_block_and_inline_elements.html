Block and inline elements

properties of

  • Block-level elements
    1. always start in a new line
    2. take up the full width available
    3. can contain both block level and inline elements
  • inline-level elements
    1. don't always start in a new line
    2. only take up as much width as necessary
    3. can only contain inline elements

  • Examples of block-level elements
    1. div
    2. p
    3. h1 to h6
  • Examples of inline-level elements
    1. em
    2. span
    3. a
    4. strong
    5. img
